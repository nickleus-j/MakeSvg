Minimal SVG Generator

A lightweight API for generating dynamic SVG assets on the fly. Use these endpoints as src attributes for your images.

Random Generation

GET/random

Generates a random SVG with default dimensions (330x250).

Example: /random

GET/random/{width}/{height}

Generates a random SVG with custom dimensions.

/random/200/200

Shapes & Grids

GET/grid/{width}/{height}

Generates a structured rectangular grid.

Example: /grid/200/200

GET/circles/{width}/{height}

Returns a canvas populated with 10 random circles.

GET/rectangle/{width}/{height}

Generates a structured rectangular grid.

Example: /rectangle/200/200

Advanced Geometry

GET/polygon/{sideCount}/

Generates a specific polygon. Try 3 for a triangle, 8 for an octagon.

/polygon/8/
GET/polygon/{sideCount}/{radius}

Generates a specific polygon. Try 3 for a triangle, 6 for a hexagon.

/polygon/6/80
GET/polygons/{count}/{sideCount}/

Generates a specific polygon. Try 3 for a triangle, 6 for a hexagon.

/polygons/4/6
GET/onion/{layers}

Generates concentric "onion" circles.

/onion/12

Utilities

GET/qr?q={url}

Generates a QR code for the provided URL string.

/qr?q=https://google.com
GET/write/{text}

Renders the provided text as an SVG graphic.

/write/Hello%20World
